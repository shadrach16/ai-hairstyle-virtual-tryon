## CONCEPT 3: "The Glow-Up Montage" ⭐ PRIMARY

### Emotional Core
Self-expression → Identity play → "I could pull off ANYTHING" → Confidence

### Scene 1 — Rapid Transformations (8s)
**Veo Prompt:**
```
Photorealistic cinematic vertical video (9:16 aspect ratio). A young Black man (late 20s, dark skin, clean shaven, bald head, strong jawline, wearing a fitted plain black t-shirt) stands centered in front of a clean white wall with soft even studio lighting. He faces the camera with a neutral confident expression. The camera is locked on a static medium close-up, centered symmetrical framing. The man slowly turns his head from profile to facing camera while making direct eye contact. Clean minimal aesthetic, fashion-editorial feel, bright soft lighting no harsh shadows. Shot on iPhone 15 Pro, photorealistic, cinematic color grade, 24fps.
```

**CRITICAL — Post-production transformation effect:**
Veo CANNOT morph hairstyles mid-clip. Build this in post:
1. Generate the base Veo clip (bald man, head turn)
2. Take **4-5 real screenshots from Hair Studio AI** showing similar face with different styles
3. In CapCut, overlay these stills at **1.2-second intervals** with quick-flash transitions (0.1s crossfade)
4. Add "whoosh" SFX + camera shake (1-2px, 0.1s) on each swap
5. Return to base Veo video for last 1.5s as "reset"

**Text overlay:** `"What if you could preview any style?"` (0-3s, typewriter animation)
**Style labels synced to each swap:** `"LOCS"` → `"FADE"` → `"BRAIDS"` → `"AFRO"` (slam-in animation, bottom 20%)

## PRODUCTION TIMELINE (5 Days)

| Day | Tasks |
|---|---|
| **Day 1** | Generate ALL 9 Veo clips (3-4 variants each = 27-36 generations). Screen record all app UI footage (3 recordings). |
| **Day 2** | Review Veo outputs. Cherry-pick best. Re-generate any failed clips. Begin rough assembly in CapCut. |
| **Day 3** | Edit Concepts 1 & 2: stitch clips, composite screen recordings, add text overlays. Build Concept 3 transformation montage from app screenshots. |
| **Day 4** | Add audio (paid versions from Epidemic Sound/Artlist, organic versions with trending sounds). Add SFX. Color-grade for consistency across clips. |
| **Day 5** | QA: watch each ad 5× on phone. Check text readability. Verify pacing. Export 9:16, 1080×1920, H.264, 30fps. Create organic audio variants. Final sign-off. |

### Veo Tips
- Run each prompt 3-4× minimum — cherry-pick the best
- Add "Shot on iPhone 15 Pro" to every prompt for natural handheld feel
- Never prompt for text-on-screen — Veo garbles text. Add in post.
- Avoid close-ups of hands with phones — frame chest-up
- Slow camera movements only (push-in, pull-back, gentle orbit)
- Use identical skin tone + clothing descriptors across all scenes for one character

### Character Consistency Strategy
1. Generate 5-10 variants of Scene 1 for each concept
2. Select the best "character" face
3. Use that frame as reference for Scene 2-3 re-generation prompts (if Veo supports img2vid)
4. If faces still drift: solve with aggressive cropping, matching color grade, and the hybrid screen-recording segments

## IMPORTANT RULES

1. **"Free" language**: NEVER use standalone "FREE." Always qualify: "Try your first look free" / "Start free" / "Try it now"
2. **Categories**: The 14 in-app categories are: Braids, Fades, Locs, Afros, Modern, Weaves, Coils, Relaxed, Bob, Straight, Protective, Traditional, Twists, Bantu Knots. "Cornrows" is a sub-style under Braids — reference as "Braids (cornrows, box braids, etc.)"
3. **Hairstyle count**: Use "100+" (safe floor). Never use the exact number in ads.
4. **Android only**: All paid targeting must be Android-only devices.
5. **Music licensing**: Paid ads = commercially licensed only (Epidemic Sound/Artlist). Organic posts = trending sounds OK.
6. **Barber portrayal**: Barbers are allies, not antagonists. The app solves communication, not skill issues.
